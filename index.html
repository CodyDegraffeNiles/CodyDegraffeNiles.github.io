Cacophony

Cacophony is a Fullstack clone of the text portion of the popular communication App, Discord. It allows a user to make an account or sign in with an existing one in order to live message other users both in servers' channels as well as through direct messages. A user can either create their own server or join an existing public server to start messaging right away. While memebers of a server can live chat, a server owner can also create/edit channels to organize the many conversations happening between members.

Mini-Xcom

Utilizing Vanila Javascript and Canvas API, MINI-XCOM allows the player to be immersed in the role of a tank commander. In turn-based combat with a reactive and descriptive user interface, the player battles traitors to the last man(tank) standing.
